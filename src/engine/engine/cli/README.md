# ISA Fabric — CLI Engine

The CLI Engine provides contributor‑facing commands for inspecting, validating, and running ISA Fabric components.

## Responsibilities

- Inspect envelopes, metrics, modulation, and governance outputs
- Validate schemas and telemetry
- Run datasets and calibration sequences
- Provide help and usage guidance

## Typical Files

- `inspect.*` — inspection commands
- `validate.*` — schema validation
- `run.*` — dataset execution
- `help.*` — CLI help
- `types.*` — command definitions
